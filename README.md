# What will you learn
- Clean architecture, micro frontend, MVVM
- Hive DB
- Bloc
- Custom toast
- Write unit test
- Simple responsive web layout
- Localization
- Dynamic theme

# Gen code

```
    flutter packages pub run build_runner build --delete-conflicting-outputs
    flutter packages pub run build_runner watch

```

# How to run Unit test

```
sh run_test.sh
```
